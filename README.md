# 📦 Projeto JDBC — Conexão Java com PostgreSQL

Projeto desenvolvido para praticar conexão direta com banco de dados 
relacional utilizando JDBC (Java Database Connectivity), sem o uso de ORM.

## 🚀 Tecnologias utilizadas

- Java
- JDBC
- PostgreSQL
- SQL

## 📚 Conceitos aplicados

- Configuração de conexão com banco de dados via `DriverManager`
- Execução de queries SQL com `PreparedStatement`
- Operações CRUD (Create, Read, Update, Delete)
- Tratamento de exceções com `SQLException`
- Boas práticas de fechamento de conexão

## 🎯 Objetivo

Entender o funcionamento da camada de persistência de dados em Java 
antes da abstração proporcionada por ORMs como JPA/Hibernate.
